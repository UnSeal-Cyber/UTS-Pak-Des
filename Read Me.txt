Cara Pemakaian :
1. Aktifkan Server.py menggunakan VSCode
2. Aktifkan Client.py menggunakan cmd (bisa lebih dari 1)
3. Chatting melalui cmd client
4. pesan akan masuk ke server

Cara Kerja :
1. Ketika dijalankan, Server akan hidup dan mengambil Ip Address secara otomatis
2. Ketika dijalankan, Client akan hidup dan mengambil Ip Address secara otobatis
3. client dan server terhubung
4. Chatting sudah dapat dilakukan
5. Client harus mengetik 'end' untuk memutuskan / disconnect

NB : penjelasan code lengkap terdapat pada source code
